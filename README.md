## Install Instructions

Clone this repository and install the plugins:

```sh
git clone https://github.com/SomeCoder99/.nvim ~/.config/SomeCoder99/.nvim
```

Open Neovim with this config:

```sh
NVIM_APPNAME=SomeCoder99/.nvim/ nvim
```
